# Py100 — Топ-100 вопросов к собеседованию Python

Современное **PWA-приложение** для подготовки к техническим собеседованиям на Python-разработчика (Middle / Senior).

Приложение работает полностью офлайн и легко устанавливается на телефон.

## ✨ Основные возможности

- 100 самых актуальных вопросов с развёрнутыми ответами
- Удобный поиск по вопросам
- Режим собеседования
- Полностью работает **без интернета**
- Устанавливается на Android как настоящее приложение
- Тёмная тема

## 🚀 Как запустить

### Рекомендуемый способ — GitHub Pages (HTTPS + PWA)

1. Запушите все файлы в репозиторий
2. Перейдите в `Settings → Pages`
3. Выберите `Deploy from branch` → `main` → `/ (root)`
4. Сайт будет доступен по адресу `https://ваш_ник.github.io/py100`
