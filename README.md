# Py100 — топ-100 вопросов к Python-собесу

Сайт + PWA-приложение для Android. Бэкенда нет: index.html + service worker.
Файлы: `index.html`, `sw.js`, `manifest.webmanifest`, `icon-192.png`, `icon-512.png`.

## Деплой

**Docker:**
```bash
docker build -t py100 .
docker run -d --name py100 -p 8080:80 py100
# или: docker compose up -d --build
```

**Без Docker:** `python3 -m http.server 8000` из папки с файлами,
либо скопировать все 5 файлов в корень статики nginx.

**GitHub Pages (рекомендую для телефона):** запушить файлы в репозиторий,
Settings → Pages → deploy from branch. Получишь бесплатный HTTPS —
он нужен для полноценной установки приложения и офлайна.

## Установка на Android

1. Открой сайт в Chrome на телефоне.
2. Меню (⋮) → «Добавить на главный экран» / «Установить приложение».
3. Готово: иконка на рабочем столе, полноэкранный режим, работает офлайн,
   прогресс сохраняется на устройстве.

Важно: офлайн и «настоящая» установка (standalone-режим) работают только
по HTTPS или на localhost. По голому http://ip:8080 будет просто ярлык
без офлайна — для этого случая проще всего GitHub Pages.

## Обновление вопросов

Все данные — массив `Q` в `index.html` (n — номер, t — тема, b — блок частоты
A–D, q — вопрос, a — развёрнутый ответ, code — пример кода). После правок
подними версию кеша в `sw.js` (CACHE = "py100-v2"), иначе телефоны будут
держать старую версию.
